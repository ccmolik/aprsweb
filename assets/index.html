APRSWeb

What?

This displays all APRS checkins my RTL-SDR has seen in the past hour.

This Golang app talks to Direwolf on a (separate) RasPI to get APRS frames, and uses Twirp to serialize those frames into JSON or protobuf (schema here) objects.

API access

Here's an example call to the API to get checkins in JSON form:

curl -X POST 'http://kd2aoy-pi.local.mesh:5000/twirp/net.sfwem.aprsweb.APRSWeb/GetCheckins' -H 'Content-type: Application/JSON' --data '{"since":1579463065997000000}'

"since" is the number of unix nanoseconds (since the epoch) specifying the earliest checkin you want to see.
